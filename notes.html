The Most Important Things to Remember

How the Web Works

The web is a collection of HTML documents. HTML stands for Hypertext Markup Language. The links between webs are called hyper links.

Elements and Tags

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. You dont have to remember all tags or attributes, just understand how HTML works and find the details when you need them. Nice list of Tags.

Inline and Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them (<DIV> is the typical example).

Avoiding Repetition

When programmers don't avoid repetition, they will often have to do the same thing over and over. This can lead to errors.

Example: Using the same class to different elements we avoid repetition.

CSS

HTML = structure and CSS = style to the structure.
DOM means Document Object Model (tree-like structure). The "tree-like structure" comes from the fact that HTML elements can have other elements inside of them. You can draw this relationship like a family tree.
Webs are made of "boxes". Boxify a design is the term used. When designing, always start with the biggest box.
